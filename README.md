# Building
### Requirements
* CMake
* Visual Studio

### Steps
1. Clone the repo ````
git clone https://github.com/neeeruuu/crosscord --recurse-submodules````
2. Open repo's folder
3. Run GenProj.bat (or open on VS Code and press Ctrl+Shift+B)
4. Open solution proj folder (crosscord.sln)
5. Build
